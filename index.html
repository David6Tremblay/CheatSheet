Semantic HTML Reference

Non-Semantic HTML

Tag Name Description Alternative Tag
<div> Division A tag that defines a division or a section in an HTML document. <section>
<span> Span A tag that is an inline container used to mark up a part of a text, or a part of a document.
<b> Bold A tag that specifies bold text without any extra importance. <strong>
<i> Italic A tag that defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic. <em>

Semantic HTML

Tag Name Description
<address> Adress Represents the contact information of a business or person.
<article> Article Represents a part of a page which is self-contained and could be published elsewhere. Common uses include blog posts or magazine articles.
<aside> Aside Represents a part of a page which is relevant to the current content, but not a part of it directly. It can be used for related links, for clarifying a statement from the current article, or even for advertising meant for the current page.
